# strace
基于strace-4.14修改，增加以下功能

## 定时统计系统调用 ##
使用-c参数统计系统调用不足之处在于，统计从strace启动到停止整个生命周期的数据，不够精细化，有时需要向vmstat/mpstat之类的定时输出结果。因此，新增-w n参数，用来指定定时输出时间，单位秒。
strace -W 1 -p 12345
